# **App Name**: CreatorKit

## Core Features:

- Image Compressor: Efficiently compress images using 'browser-image-compression' to reduce file size while maintaining visual quality.
- Smart Cropper: Utilize 'react-easy-crop' for precise image cropping with popular social media aspect ratio presets (e.g., 1:1, 9:16).
- Image Format Converter: Convert images between popular formats (WebP, PNG, JPG) using the Canvas API for optimized web use.
- AI Hashtag Generator: Generate relevant and trending hashtags using Firebase Firestore data as a tool to analyze and suggest appropriate tags.
- Progressive Web App (PWA) Support: Enable offline access and seamless mobile installation through PWA capabilities.
- Google AdSense Integration: Include placeholders for Google AdSense banners (top banner and post-action interstitial) to support monetization.
- Real-time User Feedback: Provide immediate success notifications via toasts and indicate processing states with loading spinners for all user interactions.

## Style Guidelines:

- Background: A deep, professional charcoal grey (#16181C) to underpin the modern dark theme.
- Primary UI color: A clear and bright sky blue (#26A0DA) for main interactive elements, such as buttons and selected states.
- Highlight accent: A soft, refreshing seafoam green (#6AE1CC), chosen for its complementary contrast and analogy to the primary, ideal for sub-headings or subtle distinctions.
- Vibrant accent 1: Pure Neon Cyan (#00f7ff) as a bold accent for key call-to-actions and impactful visuals, as explicitly requested.
- Vibrant accent 2: Crisp Lime Green (#32CD32) for success indicators, alerts, and other distinct elements, also explicitly requested.
- All text: 'Space Grotesk' (sans-serif) for a modern, techy, and precise feel consistent across the application.
- Utilize 'lucide-react' icons throughout the application to maintain a clean, modern, and consistent visual aesthetic.
- Implement a responsive, mobile-first grid layout for efficient tool selection and navigation, ensuring optimal usability across all device sizes.
- Incorporate subtle loading animations and concise success toasts to provide clear and immediate feedback to users during all interactions.
